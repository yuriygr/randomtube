## Структура папок

Обычно, в данном бойлерплейте оно выглядит так

```
+-- app
|  +--  components
|  |  -- filters
|  |  -- global_ui
|  |  -- router
|  |  -- store
|  |  -- i18n
|  +--  services
|  |  -- api
|  |  -- bus
|  |  -- storage
|  |  -- utilities
|  +-- locales
|  |  -- ru.json
|  |  -- en.json
|  |  etc

+-- components
|  +-- _global
|  |  +-- layers
|  |  +-- page
|  |  -- Icon.vue
|  +-- _site
|  +-- modals
|  etc


+-- Views
|  -- Home.vue
|  -- Help.vue
|  etc
```
